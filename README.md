# Child Development Tracker

Track your children's development -- goals, schedules, milestones, and daily logs.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Features

- **Dashboard** -- Overview of all children with stats, today's schedule, goal progress
- **Goals** -- Track academic, social, physical, creative, and life-skills goals with progress bars
- **Weekly Schedule** -- Plan daily activities by category (school, play, rest, etc.)
- **Milestones** -- Record achievements on a visual timeline
- **Daily Log** -- Track mood, sleep, highlights, and notes

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS, TanStack Query, Wouter, Lucide Icons
- **Backend**: Express, better-sqlite3
- **Database**: SQLite (auto-created as `tracker.db`)

No PostgreSQL, no Docker, no config needed.
